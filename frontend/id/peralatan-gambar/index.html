CARA MENGGUNAKAN

Tiga Langkah Mudah

1

Masukkan gambar kamu

Seret dan lepas atau klik untuk mengunggah file JPEG, PNG, WebP, SVG, atau GIF hingga 15 MB per file.

2

Pilih pengaturan kamu

Pilih tab — Kompres, Konversi, Ubah Ukuran, atau Watermark — lalu sesuaikan kualitas, format, dimensi, atau opsi watermark.

3

Unduh hasilnya

Unduh setiap file satu per satu atau ambil semuanya sebagai satu ZIP. Semua pemrosesan terjadi di browser kamu — tidak ada yang diunggah.

FAQ

Pertanyaan yang Sering Diajukan

Apakah gambar saya diunggah ke server?

Tidak. Semua operasi — kompresi, konversi, ubah ukuran, dan watermark — berjalan sepenuhnya di dalam browser kamu menggunakan Canvas API. File kamu tidak pernah meninggalkan perangkat, dan tidak ada yang dikirim ke server manapun. Kamu bahkan bisa memutus koneksi internet setelah memuat halaman ini dan alatnya tetap berfungsi.

Berapa banyak gambar yang bisa saya kompres sekaligus?

Tidak ada batasan. Kamu bisa memasukkan gambar sebanyak yang kamu mau dan akan diproses satu per satu. Setiap file bisa sampai 15 MB. Untuk batch yang sangat besar, waktu pemrosesan tergantung kecepatan perangkat kamu.

Bagaimana cara mengompres gambar ke ukuran tertentu seperti 100 KB?

Aktifkan toggle Ukuran Target di tab Kompres, lalu masukkan ukuran file yang kamu inginkan dalam kilobyte (misalnya 100). Alat ini akan menyesuaikan kualitas secara berulang sampai file output sama dengan atau di bawah ukuran target kamu.

Format apa yang terbaik untuk gambar web di 2026?

WebP menawarkan keseimbangan terbaik antara kualitas, kompresi, dan dukungan browser. AVIF menghasilkan file yang lebih kecil tapi encoding lebih lambat dan dukungan browser masih berkembang. Untuk kompatibilitas maksimal, gunakan WebP dengan fallback JPEG. Tab Konversi kami memudahkan kamu untuk batch-konversi gambar ke format manapun.

Bagaimana cara mengompres gambar secara gratis?

Buka Peralatan Gambar, seret dan lepas gambar kamu ke zona drop, pilih level kualitas dan klik Unduh. Semua pemrosesan berjalan di browser kamu, gratis, tanpa daftar dan tanpa upload file.
