iPhone पर Twitter वीडियो कैसे download करें?

Twitter खोलें, वीडियो वाला tweet ढूंढें, Share बटन पर टैप करें और "Copy Link" चुनें। इस page पर आएँ, ऊपर दिए बॉक्स में लिंक paste करें और Download पर टैप करें। वीडियो आपके iPhone की camera roll में save हो जाएगा। कोई app ज़रूरी नहीं — iPhone पर Safari या Chrome में सीधे काम करता है।

Twitter GIF कैसे save करें?

ऊपर "GIF" टैब पर क्लिक करें, GIF वाले tweet का लिंक paste करें और Download पर क्लिक करें। हमारा tool Twitter GIF को downloadable file में convert करता है। Twitter GIF असल में छोटे MP4 वीडियो होते हैं, इसलिए downloaded file MP4 होगी जो GIF की तरह loop करती है।
